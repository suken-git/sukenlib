・MapChipEditorとは。
70期photonがSukenLibを使って作った、ゲームのマップ作成を目的とした、
画像を並べるためのツールです。フリー素材として配布されている、
マップチップが敷き詰められている画像などを読み込んで、使うことができます。
MCEは数研ライブラリに（勝手に）埋め込まれており、
MCEクラスから、ファイルの中身を参照することができます。
（今のところ座標とレイヤーからチップに対応した数字を読み込めるぐらい）

・使い方

　・初期設定
　　MCE-custom.exeを使って、
　　チップを敷き詰めるパレットのサイズ、読み込むチップのpixel幅、
　　レイヤーA,B,Cそれぞれで読み込む画像、軽量化するかどうかを設定します。

　　数字や文字をクリックすると、キーボード入力できるようになります。
　　使える文字は数字とアルファベット、アンダーバーのみです。
　　Enterで確定。Escapeでキャンセルします。
　　OKをクリックするか、キーボード入力をしていないときに
　　Enterで、設定を保存することができます。

　　WIDTH　パレットのpixel幅
　　HEIGHT　パレットのpixel高さ
　　EDGE　チップ一つの一辺のpixel幅
　　GRAPH_A,B,C レイヤーごとに読み込む画像
　　WeightSaving　軽量化設定。YESだと透過画像が透過しなくなる。

　　これらの設定は、作成するファイルごとに変えることができますが、（軽量化設定は別）
　　新規作成時点では、ここでの設定になります。
　　ここで設定した内容は、mcepic内のcustom.mceに保存されるので、
　　消さないようお願いします。

　・ファイル選択画面
　　MCE.exeを起動すると、まずこの画面になります。
　　からの場所をクリックすると、新しくファイルが作成されます。
　　からのファイルはクリックすると、名前を入力することができます。

　　ここでできる操作は
　　・名前変更
　　・場所入れ替え
　　・ファイルの削除
　　・スクロール

　　名前をクリックすることで、そのファイルを編集できます。

　・編集画面

　　Spaceでメニューバーの表示切替ができます。
　　マウスホイールもしくはCtrl + ＋、Ctrl + ー　で拡大縮小。
　　中クリックでパレットをつかんでスクロールできます。
　　（上下左右キーでもスクロールできる）
　　
　　・レイヤー
　　　文字を左クリックで書き込みレイヤーの変更
　　　白いところを左クリックで、表示レイヤーの変更。
　　　右クリックすると、そのレイヤーを書き込むレイヤーにして、
　　　そのレイヤーだけを表示するようにします。

　　・チップ選択
　　　画像が正しく読み込まれていたら、上の８つ横に並んだ枠に、チップが表示される。
　　　左クリックでチップを選択。
　　　横の矢印でスクロール。左クリックで一つづつ、右クリックで8つずらす
　　　右のgraphで画像から選択できる。左クリックで選択。
　　　レイヤーごとに選択チップは保存される。
　　
　　・ツール類
　　　左クリック、右クリックそれぞれにツールの割り当てができます。
　　　ツール使用中に、反対をクリックすると、キャンセルできます。
　　　・ペン　クリックで選択チップをおく。
　　　　　　　長押しで広域。
　　　・消しゴム　クリックでチップを消す。
　　　　　　　ペンと同様、長押しで広域。
　　　・移動　選択しているチップを動かす。
　　　・バケツ　ペイントツールにおけるバケツ。
　　　　　　　　クリックしたチップと選択チップが同じだと、消去する。
　　　・範囲選択ツール　選択したチップのみツールの効果を受ける。
　　　　　　　　　　　　クリックで全選択。長押しで範囲選択。
　　　　　　　　　　　　Ctrl、Shiftを押しながらだと変わり、
　　　　　　　　　　　　選択範囲を移動できたりと、多機能（疲れた）
　　　・スポイト　スポイト

　　・作業を元に戻す、進める。
　　　左上の矢印から。

　　・設定
　　　右上のcustomから。

　　・ファイル選択画面に戻る
　　　右下のBACK

　　・保存する
　　　右下　SAVE

　・ショートカットキーについて。
　　Ctrl　＋　S　：　保存
　　Ctrl　＋　A　：　全選択
　　Ctrl　＋　C　：　コピー
　　Ctrl　＋　X　：　切り取り
　　Ctrl　＋　V　：　貼り付け（長押し状態でマウスをずらして場所を決める）
　　Ctrl　＋　Z　：　作業を元に戻す
　　Ctrl　＋　Y　：　作業を進める
　　Delete　：　選択レイヤー上のチップを全削除
　　　　　　　　　　　　
　　コピー＆ペーストでは一応レイヤー間の移動ができる
